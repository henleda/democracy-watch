# AI Package

Bedrock-powered AI agents.

## Agents
- discovery/: Find content sources
- extraction/: Get transcripts
- classification/: Extract promises (Claude)
- alignment/: Analyze vote-promise (Claude Haiku)
- embeddings/: Generate vectors (Titan)

## Models
- Claude Sonnet: Complex analysis ($3/$15 per 1M)
- Claude Haiku: Bulk classification ($0.25/$1.25 per 1M)
- Titan Embeddings: 1024-dim vectors
